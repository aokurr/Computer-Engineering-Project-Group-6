What We Do

We are a team aims to develop a semi-autonomous robotic dog capable of remotely searching for objects in unstructured environments, generating 3D maps of its surroundings, and providing real-time data through network communication and a dedicated app.

Autonomous movement and image processing for object recognition

Autonomous movement algorithms are used for our robot to move independently of the user and image processing algorithms are used so that our robot can move smoothly on the path using the images we get from the camera on the robot.

Network communication and app integration for remote control and data visualization

Communication protocols are used for remote monitoring and control of the robot’s movements and actions through a computer or other device.

3D mapping for environmental awareness and navigation

The robot dog is equipped with a camera to capture images of the environment and use these images to create a 3D map of the environment by utilizing computer vision algorithms.

Desktop application for remote operations

The desktop program enables users to remotely operate the robot, analyze and evaluate the data obtained from the robot’s sensors, and monitor the real-time video feed from the robot’s camera.
